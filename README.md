# AgroTech:AI

## Problem Statement

Farmers are getting a lot loss due lack of knowledge on type of crop need to be grown in an area, how the diseases can be cured for a different spicies of crops and it leads to the crisis and less yeilding.

Also Farmers are not aware of how the current market rates of crops and government schemes that they are eligible.

To fix this problem I have created a mobile application that gives solution to all these problems.

## Features

- To create a feature that analyze the disease on crop based on the image(Accurately).
- To create a feature that predicts the perfect crop for a region.
- To create a feature that provides the current market trends to farmer.
- To create a feature that provides governemnt schemes that are applicable for that farmer.
- To create a feature that can voice chat with farmers in any language.
- To create a feature that user can set required language.

## Tech Stack

- Frotnend: Flutter
- Backend: Node, Express, python
- Database: MongoDB, PineconeDB, FirebaseDB
- Other: LLM, NLP, ML
